Sound Flower

Creative Coding / UI Case Study

Explores sound visualization through the means of a mobile application which records and stores sound and its corresponding imagery generated through code. Users are able to create their own "sound flower" by collecting or uploading their own sound.

Hawker Sensor

UI / UX Case Study

Hawker Sensor is a case study that utilizes the data set from the National Environment Agency in Singapore that measures information about Hawker centers, which are public food courts in Singapore. The case study analyzes two sources and conceptualizes a location based mobile application that allows the selected target audience to keep track of their culinary experience in Singaporean hawker centers.
